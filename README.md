Userscripts
===

Various FireMonkey scripts I've written for use on various websites to do various things.

I've only tested them on my machine running Firefox, so they may or may not work on yours. I also can't guarantee that they're all compatible with other userscript managers (like Tampermonkey or Violentmonkey).

If they do, great! If they don't, oh well.

The 'before2026' branch is the old version of this repository and is not updated anymore.
